# Club-and-Society-Management-Portal.
A web-based platform designed to streamline the management of clubs, societies, and student organizations. This portal provides administrators, members, and event coordinators with tools to efficiently handle memberships, events, communication, and resources

Overview

The **Club & Society Management Portal** is a full-stack web application designed to simplify and digitalize the management of student clubs and societies. It provides a centralized platform for students and administrators to interact, manage activities, and stay updated with events and announcements.



1.Features

User Features

* User Registration & Login Authentication
* Join and Leave Clubs
* View Available Clubs
* Access Events and Announcements
* Responsive and user-friendly interface

2.Admin Features

* Manage Clubs (Create, Update, Delete)
* Manage Events and Announcements
* View and Manage Student Registrations
* Role-based access control


3.Tech Stack 

Frontend

* HTML
* CSS
* JavaScript

Backend

* PHP
* Java

Database

* MySQL


System Architecture

The application follows a multi-layered architecture:

-Presentation Layer: HTML, CSS, JavaScript
-Application Layer: PHP & Java for handling logic and requests
-Data Layer:MySQL database for storing user, club, and event data
